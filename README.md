# E-commerce Sales and User Behaviour Analysis

## Project Overview

This project analyses e-commerce sales, user behaviour and conversion-related patterns using SQL, Python, pandas, statistical testing and Tableau. The goal is to transform raw event and transaction data into structured insights that can support business and product decisions.

## Business Questions

- Which channels, geographies and devices contribute most to sales and user activity?
- How do user sessions, registrations and orders differ across key segments?
- Are there meaningful patterns in sales dynamics and customer behaviour?
- What data quality issues should be checked before making business conclusions?

## Tools and Technologies

- SQL
- Python
- pandas
- scipy / statsmodels
- Tableau
- Jupyter / Google Colab

## Methods Used

- Data loading and initial inspection
- Data quality checks
- Data cleaning and preparation
- Joins and aggregations
- Grouped analysis by channel, country, device and user behaviour
- Pivot tables and summary metrics
- Exploratory data visualisation
- Statistical analysis and significance testing
- Tableau dashboard preparation
